Olympic Games Dashboard with SQL and Power BI

26 February, 2023

I loved creating this dashboard; I used historic Olympic data starting in 1896 to create a lovely dashboard with colour-coding to match the Olympic medal levels. I cleaned and prepared the data in SQL Server, imported it into PowerBI, and created the dashboard using donut charts, bar charts, line charts and KPI's. Then, I got a bit more curious about the data and explored what the data can tell us about Ireland and women's boxing. You might find the results as interesting as I did!
